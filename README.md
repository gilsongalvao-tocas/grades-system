📚 Project Overview
This project demonstrates the use of Object-Oriented Programming (OOP) concepts in Python while working with common data structures such as:
Lists
Tuples
Dictionaries
Sets
The program creates and manages student records, calculates averages, validates emails using regular expressions, and performs several list and tuple operations.
🎯 Objective
The goal of this assignment is to practice:
Creating Python classes and objects
Using methods and attributes
Working with lists, tuples, dictionaries, and sets
Applying OOP principles
Using regular expressions for email validation
Handling exceptions with try/except
